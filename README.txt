
Rock Paper Scissors Game  
=========================

### Introduction
Welcome to my Rock Paper Scissors game! This project is written in Python and serves as a fun way to play the classic game against a computer opponent.  

### Features
- Play Rock, Paper, or Scissors against the computer.  
- Randomized computer choices for fair gameplay.  
- Immediate results after each round.  

### How to Play
1. Run the Python script.  
2. Press a button (Rock, Paper, or Scissors) when prompted.  
3. The game will display the winner of the round.  
4. Play as many rounds as you'd like!  

### Requirements
- Python 3.x installed on your computer.  

### Installation
1. Clone this repository or download the Python script.  
2. Run the script using your Python interpreter or open dist folder and open main.exe.
